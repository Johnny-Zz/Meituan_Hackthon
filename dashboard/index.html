No-regression Gate 规则

  • 必须通过 Python 编译和静态审查；禁止新增 requests/openai/deepseek/socket/subprocess 等危险依赖进入 solver.py。
  • 所有本地 case 必须 valid，任务/骑手重复、非法边都会被拒绝。
  • lower-is-better 分数不得高于 patch 前基线；退化则自动恢复 pre-patch solver。
  • 没有 DEEPSEEK_API_KEY 时使用本地 safe fallback，仍然完整跑通 patch、diff、gate 与日志闭环。
